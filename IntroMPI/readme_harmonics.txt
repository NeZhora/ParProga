# MPI: Сумма гармонического ряда

## Описание
Параллельное вычисление S = Σ(1/n) для n от 1 до N.

## Метод распараллеливания
Используется **циклическое (round-robin) распределение**: процесс с рангом `r`
суммирует элементы с индексами r+1, r+1+size, r+1+2·size, ...

Это обеспечивает равномерную нагрузку: каждый процесс обрабатывает примерно N/size
элементов.

## Сбор результатов
Используется **MPI_Reduce** с операцией **MPI_SUM** — коллективная операция,
которая суммирует локальные результаты всех процессов и помещает итог на процесс 0.
