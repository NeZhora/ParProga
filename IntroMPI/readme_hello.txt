# MPI Hello World

## Описание
Простейшая MPI-программа. Каждый процесс выводит приветствие, свой ранг и общее
число процессов.

## Ключевые концепции
- **MPI_Init** — инициализация среды MPI. Должна быть вызвана до любых других
  MPI-функций.
- **MPI_Comm_rank** — возвращает уникальный номер (ранг) процесса в коммуникаторе.
- **MPI_Comm_size** — возвращает общее число процессов в коммуникаторе.
- **MPI_COMM_WORLD** — коммуникатор по умолчанию, объединяющий все процессы.
- **MPI_Finalize** — корректное завершение среды MPI.

## Компиляция и запуск
```bash
mpicc -o mpi_hello mpi_hello.c
mpirun -n 4 ./mpi_hello
