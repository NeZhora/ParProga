# MPI: Круговая пересылка

## Описание
Переменная типа int передаётся по кольцу: 0 → 1 → 2 → ... → (size-1) → 0.
Каждый процесс инкрементирует значение на 1.

## Ключевые концепции
- **MPI_Send / MPI_Recv** — блокирующие операции «точка-точка».
- **Кольцевая топология**: следующий = (rank+1) % size.
- Процесс 0 инициирует передачу и замыкает кольцо, принимая от последнего.

## Важно
Порядок Send/Recv критичен для избежания **deadlock** (взаимной блокировки).
Процесс 0 сначала отправляет, потом принимает. Все остальные — сначала принимают,
потом отправляют. Это гарантирует отсутствие тупиков.

mpicc -o mpi_ring mpi_ring.c
